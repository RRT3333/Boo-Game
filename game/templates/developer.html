마크다운 파일 스캔

모든 마크다운 파일을 스캔하고 메타데이터를 생성합니다:

python manage.py scan_markdown_files

이 명령은 모든 마크다운 파일을 찾아 메타데이터를 생성하고, 자동으로 문서 허브에 표시합니다.

심볼릭 링크 생성

마크다운 파일을 static 폴더에 심볼릭 링크로 생성합니다:

python manage.py create_md_symlinks

Windows에서는 관리자 권한이 필요할 수 있으며, 실패 시 파일 복사가 수행됩니다.
