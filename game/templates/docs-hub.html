    마크다운 파일 관리

    마크다운 파일을 static 폴더에 심볼릭 링크로 생성하려면 다음 명령어를 실행하세요:

    python manage.py create_md_symlinks

    이 명령어는 프로젝트 내의 모든 .md 파일을 찾아 static 디렉토리에 동일한 구조로 심볼릭 링크를 생성합니다.

    Windows 시스템에서는 관리자 권한이 필요할 수 있으며, 심볼릭 링크 생성이 실패하면 자동으로 파일을 복사합니다.
